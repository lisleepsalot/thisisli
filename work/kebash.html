My Role
I was responsible for character design, 3d modeling, and UI elements designs.
Project Overview
Kebash! is a controller-based PvP game built with Unity by Brown RISD Game Developers in 2022 fall semester.
The game is available here.
The logo
I customized a tyfaface for Kebash! This typeface features thick strokes, compacted bowls, and slight top-down perspective distortion, and it aims to be used flexibly in different settings (title, heading, body text, symbols, etc). The distortion is intended to align with the top-down viewpoint of the game.
UI Elements
I have created a series of UI textures based on the color palette of the typefaces and the joyful atmosphere of the game. Illustrations in instruction pages by Vivian Chan.
Character Design
The characters in the game are intended to be represented by empty kebab sticks. To simplify the animation process, I have altered the shape of the kebab sticks to a ball shape and a cylinder shape. Despite their different appearances, they conform to a standard proportion and outline.
Character Modeling
Character modeled by blender: a takoyaki character, a meat ball character, and the basic kebab stick.
